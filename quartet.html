Quartet d'Anscombe

Cette infographie a été réalisée dans le cadre du cours de visualisation de LP MIND.
L'objectif de ce projet était d'expliquer le Quartet d'Anscombe à l'aide d'une infographie.

Pour réaliser cette infographie, j'ai utilisé Illustrator afin d'être assez libre dans la création.

J'ai essayé de définir un code couleur, afin que les personnes puissent associer en un coup d'oeil les séries de données à leur représentation graphique.

Néanmoins, il y a un problème de couleur sur la phrase "un ensemble de", car le bleu utilisé n'est pas le bon. On peut voir la différence avec les autres éléments bleus de l'infographie.
